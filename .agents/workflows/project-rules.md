# Regras do Projeto

## Testes e Verificação
- **NÃO** usar o browser_subagent para testar a aplicação. Nunca interagir com a DOM.
- **NÃO** clicar, preencher formulários, ou navegar pela aplicação via automação.
- Se precisar verificar algo visualmente, **peça ao usuário para testar e relatar erros**.
- Apenas descreva as mudanças feitas e o que o usuário deve verificar manualmente.
- Screenshots e gravações de tela pelo browser_subagent não são desejados.
